HTML/CSS:

Fundamentalne technologie tworzenia Aplikacji internetowych pomimo wielu lat nadal stanowią nieozowną cześć każdej witryny internetowe, w ciągu tych lat wielokrotnie usprawniane i unowocześniane. Aktualny HTML 5 wprowadził semantyczny HTML który ułatwia prace wyszukiwarce i pozwala na lepszą organizacje kodu. dobra ich znajomość jest niezbedna do tworzenia dobrych witryn WWW.

JavaScript

Skryptowy jezyk programowania używany przez wszystkie przeglądarki Internetowe. stworzony w 1997 przez firmę Netscape. Jezyk który wprowadził rewolucje w tworzeniu uwczesnych stron internetowych,gdyby nie JavaScript internet nie byłby tym czym jest dziś wysoce wydajny i olbrzymich możliwościach stosowany jest do pisania logiki Aplikacji.

ES6

Lub też ECMaScript 6 to popularny i nowoczesny standard jezyka JavaScript, wprowadził on wiele usprawnień takich jak Klasy, funkcje Strzałkowe itp.

Node JS

Technologia pozwalająca na uruchomienie JavaScript po stronie serwera

Sass/Scss

Prekompiler jezyka opisowego CSS znacznie rozszerzający jego możliwości o właściwości charakterystyczne dla jezyków takich jak javaScript czy C++

TypeScript

Nadzbiór jezyka javaScript stworzony przez firmę Microsoft udostepniony na darmowej licencji Opensource. wprowadza on statyczne typowanie i wiele innych możliwości znacznie zwiekszające możliwości programisty

Angular

Framework JS stosowany do pisania pelnoprawnych aplikacji internetowych typu SPA (Single Page Aplications) storzony przez Google udostepnia on ogromne możliwości tworzenia Aplikacji z wykorzystanie TypeScript. jeśli potrzebujesz dużej aplikacji Angular napewno cię nie zawiedzie.

React JS

Framework stworzony przez programistów facebooka pozwalający łatwo i szybko tworzyć funkcjonalny i przejrzysty interfejs użytkownika
